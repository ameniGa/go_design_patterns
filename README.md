 1. Behavioral design patterns:
    1. Strategy
    2. Observer
    3. Command
    
 2. Creational design patterns:   
    1. Factory method
    2. Abstract factory
    3. Singleton
    
 3. Structural:
    1. Adapter
    2. Decorator
    3. Facade
