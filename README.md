# Design Patterns in Golang

 A list of design patterns implemented in Go with simple and stupid examples:
 
 1. Behavioral design patterns:
    1. Strategy
    2. Observer
    3. Command
    4. Chain of Responsibility
    5. Mediator
    6. Memento
    7. State
    8. Template
    9. Visitor
    10. Iterator
    
 2. Creational design patterns:   
    1. Factory method
    2. Abstract factory
    3. Singleton
    4. Prototype
    5. Builder
    
 3. Structural:
    1. Adapter
    2. Decorator
    3. Facade 
    4. Composite
    5. Flyweight
    6. Proxy
    7. Bridge
